# BCT(BlockChain Transaction)

区块链支付结算系统的设计与实现。

## 环境要求

* python3
* python-flask
* Crypto

## 使用

### 环境安装

1. 安装[python3](https://www.python.org/)
2. 安装依赖

```bash
pip install -i http://pypi.douban.com/simple -- trusted-host pypi.douban.com -r requirements.txt
```
